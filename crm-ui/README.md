# CRM user interface

## Getting started

1. Install `npm`

2. Install `gulp`

   ```sh
   npm install --global gulp-cli
   ```

3. Configure the different environments

   ```sh
   cp config.example.js config.js
   ```

   and edit the paths and user as necessary

4. Install dependencies

   ```sh
   npm install
   ```

5. Start the servers

   Make sure the crm-server is running if working on the dev environemnt!

   ```sh
   gulp server
   ```
