Customer Management 
===================

Features
--------

    - Web user interface 
      - Add / View / Update / Delete Contacts
      - Add / View / Update / Delete Accounts
      - View Activities related to a Contact or Account
      - Add / View Notes to Contact 
      - Add / View Documents to Contact 
    
    - REST API feature matches Web UI
